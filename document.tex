\section{Experiment Design}
    O "experimento" deste projeto consiste na validação da eficácia do chatbot. Deve ser estruturado da seguinte forma:
    \begin{itemize}
        \item \textbf{Ambiente de Teste}: Servidor Flask local, base de conhecimento do IFSC Garopaba.
        \item \textbf{Metodologia de Teste}: Aplicação de um conjunto de perguntas reais (secretaria) vs respostas geradas pelo bot.
        \item \textbf{Métricas}: Acurácia (o bot entendeu a intenção?), Precisão (a resposta estava correta?) e Tempo de Resposta.
        \item \textbf{Grupo de Avaliadores}: Teste cego com estudantes ou servidores para avaliar a utilidade percebida.
    \end{itemize}

\section{Tutorial 1: Configuração do Ambiente Flask}
\label{sec:tut1}

\begin{boxTutorial}{Configuração Inicial}
Para iniciar o projeto, a aluna deve configurar um ambiente virtual Python isolado e instalar as dependências básicas.

\begin{itemize}
    \item Criar ambiente virtual: \texttt{python3 -m venv .venv}
    \item Ativar ambiente: \texttt{source .venv/bin/activate}
    \item Instalar pacotes: \texttt{pip install flask flask-cors python-docx fpdf}
\end{itemize}

\begin{minted}[bgcolor=gray!5, frame=single]{python}
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/chat', methods=['POST'])
def chat():
    user_msg = request.json.get('message')
    return jsonify({"response": f"Você disse: {user_msg}"})

if __name__ == '__main__':
    app.run(debug=True)
\end{minted}
\end{boxTutorial}

\section{Tutorial 3: Automação e Geração de Documentos}
\begin{boxTutorial}{Geração de Docs}
Como gerar um "Atestado de Matrícula" ou similar automaticamente.
Usaremos a biblioteca \texttt{python-docx} para carregar um modelo \texttt{.docx} e substituir as variáveis.

\begin{minted}[bgcolor=gray!5]{python}
from docx import Document

def gerar_documento(nome_aluno, curso):
    doc = Document('template_secretaria.docx')
    for p in doc.paragraphs:
        if '{{NOME}}' in p.text:
            p.text = p.text.replace('{{NOME}}', nome_aluno)
    doc.save(f'atestado_{nome_aluno}.docx')
\end{minted}
\end{boxTutorial}
